# UFC-DASHBOARD🥊

## Descrição
O UFC-DASHBOARD é um dashboard interativo feito no power BI com o auxilio da biblioteca pandas do python, para a analise de dados de um dataset. Nesse dashboard foi utilizado um dataset de dados com todas as lutas e cards do evento Ultimate Fighting Championship(UFC), com a finalidade de mostrar de forma interativa dados sobre o UFC.

## Pré-requisitos📍

Para a vizualização completa do dataset, é necessario a instalação previa do aplicativo "Power bi desktop" no site da microsoft, com o aplicativo instalado, baixe o arquivo "ufc-dashboard.pbix" presente nesse repositorio, apos isso execute o arquivo no Power BI desktop e tenha a visão completa dos principais dados do UFC na palma da sua mão.

Para a vizualização da analise previa dos dados, feita com a ajuda da biblioteca pandas no python, é necessario a utilização de uma IDE para poder executar o arquivo "Analise_pandas.ipynb" 

IDEs recomendados:

- Google colaboratorio
  
  Baixe o arquivo "Analise_pandas.ipynb" presente neste repositorio, em seguida abra o arquivo "Analise_pandas.ipynb" no google colaboratorio e rode todos os espaços de codigo

- Visual studio code com a instalação da extensão "jupyter notebook"
  
  abra o visual studio code, na area de extensões baixe a extensão "jupyter", em seguida baixe o arquivo "Analise_pandas.ipynb" presente neste repositorio, dentro do visual studio code abra o arquivo "Analise_pandas.ipynb" e execute todos os espaços de codigo 

## Cuidados❗
Para o funcionamento total do dashboard, é de extrema importancia que o arquivo seja interpretado como um "arquivo.PBIX", para funcionar diretamente no power BI desktop

Procure não fazer alterações bruscas nos codigos da analise previa, para não prejudicar seu funcionamento

Em caso de dificuldades na execução do codigo no visual studio code, tente abrir um espaço de codigo e tente rodar o comando "pip install pandas", para a instalação da biblioteca pandas do python

## Funcionalidades🥋
O dashboard feito em power bi contam com funções interativas para deixar os amantes de MMA sempre ligados nas principais informações de toda a historia do UFC e de seus lutadores, as principais funcionalidades do dashboard são:

-Caixa de texto, responsavel por fazer a busca de dados relacionados pelo nome do lutador

-Limpador de filtros, para voltar ao dashboard padrão

-Graficos interativos contendo informações sobre:

  - grafico com a quantidade de lutadores por categoria de peso
  
  - Ranking de lutadores por metodo de vitoria
  
  - Cards com o vencedor do confronto
  
  - Grafico com a data e nome do evento, vencedor e perdedor do confronto
  
  - Cards com o tempo total da luta
  
  - Cards com o round em que o confronto foi finalizado
  
  - Cards com o etodo de vitoria da luta
